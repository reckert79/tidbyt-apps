# Tennis Match Finder

A SwiftUI iOS app to help tennis players find matches in their local area.

## Features

- Post availability and find matches
- - Court discovery with MapKit
  - - Weather integration
    - - Friends and groups
      - - Player ratings and trust scores
        - - Court check-in and live busyness
         
          - ## Setup
         
          - 1. Open in Xcode
            2. 2. Build and run on iOS simulator or device
               3. 3. Requires iOS 17+
